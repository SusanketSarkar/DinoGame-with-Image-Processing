This project was built using:
- Python 3.9.6
- Pygame
- TensorFlow 
- Keras
- OpenCV

To run this project you can 
  - clone it using ```$ git https://github.com/SusanketSarkar/DinoGame-with-Image-Processing.git ```
  - run ```main.py``` from root: ```$ py main.py ```
